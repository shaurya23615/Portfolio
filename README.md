This is a **personal portfolio website** designed to showcase skills, projects, and contact information. It is fully responsive and interactive.  

---

## Features

- Responsive design for desktop and mobile devices
- Sections: Home, About, Skills, Portfolio, Contact
- Interactive buttons and smooth section transitions
- Multi-color background effect for a modern look
- Contact form integration

---

## Technologies Used

- HTML
- CSS (Flexbox, Grid, Animations)
- JavaScript (DOM manipulation, form validation)
